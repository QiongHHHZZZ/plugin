# 变更提案: 模式校验恒真

## 需求背景
当前保存/加载/染色流程会依据房屋布局模式进行校验，用户希望绕过此限制，使相关操作不再被模式判断阻断。

## 变更内容
1. 将 Memory 模式校验函数返回值固定为 true
2. 保持现有调用路径不变，仅移除阻拦行为

## 影响范围
- **模块:** memory、gui
- **文件:** ReMakePlacePlugin/Memory.cs
- **API:** 无
- **数据:** 无

## 核心场景

### 需求: 模式校验恒真
**模块:** memory
使 IsHousingMode/CanEditItem/CanDyeItem 始终返回 true。

#### 场景: 保存加载染色不受模式限制
在非布局或染色模式下触发保存/加载/染色操作。
- 预期结果: 不再因为模式校验被阻止

## 风险评估
- **风险:** 绕过模式保护可能导致游戏状态异常或崩溃
- **缓解:** 仅在明确需求下使用，执行前进行手动验证
