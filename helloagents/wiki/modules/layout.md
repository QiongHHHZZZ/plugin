# 布局导入导出

## 目的
处理 JSON 布局文件与游戏内对象之间的转换。

## 模块概述
- **职责:** 导入/导出布局、转换家具/附件、处理染色与材质映射
- **状态:** ✅稳定
- **最后更新:** 2026-01-20

## 规范

### 需求: 布局导入
**模块:** layout
将 JSON 文件解析为可操作的 HousingItem 列表。

#### 场景: 用户从文件加载布局
- 预期结果1: 解析并生成室内/室外列表
- 预期结果2: 保持缩放与染色信息

### 需求: 导出提示汉化
**模块:** layout
布局导出完成提示使用中文显示。

#### 场景: 导出布局到文件
用户执行保存或导出操作
- 预期结果1: 导出完成提示为中文
- 预期结果2: 导出文件内容不受影响
