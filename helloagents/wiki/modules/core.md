# 核心控制

## 目的
负责插件生命周期、命令入口、布局保存/加载/匹配、应用摆放与染色流程。

## 模块概述
- **职责:** 初始化/释放资源、注册命令与 Hook、管理任务队列与家具列表
- **状态:** ✅稳定
- **最后更新:** 2026-01-20

## 规范

### 需求: 布局保存与加载
**模块:** core
提供保存/加载/应用入口，协调 Memory 与 Layout 管理器。

#### 场景: 用户在插件界面点击保存或加载
进入房屋布局相关模式
- 预期结果1: 调用保存/加载流程并输出日志
- 预期结果2: 更新当前布局匹配状态

### 需求: 布局匹配空对象保护
**模块:** core
当无法获取对象列表时，布局匹配应安全退出。

#### 场景: 非布局模式打开配置窗口
不处于房屋布局相关场景
- 预期结果1: 不抛出异常
- 预期结果2: 界面可正常渲染

### 需求: 室外地块缺失保护
**模块:** core
室外地块数据缺失时不应抛出异常。

#### 场景: 室外匹配或摆放时地块未配置
地块编号不在内置映射中
- 预期结果1: 不抛出异常
- 预期结果2: 相关流程安全退出

### 需求: 聊天提示汉化
**模块:** core
聊天栏提示与错误信息使用中文显示。

#### 场景: 执行保存/加载/染色
用户触发保存、加载、摆放或染色流程
- 预期结果1: 聊天栏提示为中文
- 预期结果2: 错误信息可读且包含关键信息
